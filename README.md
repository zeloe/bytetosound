# bytetosound
 Reads raw data and outputs it as a wave file or image.
## How to use

### edit in .cpp files path to input and output 
Just edit empty strings.

## How to build
> g++ -std=c++17 -o outputfile  input.cpp 

Example

>  g++ -std=c++17 -o wavefilecreator folder_to_wav.cpp
