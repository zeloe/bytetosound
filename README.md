# bytetosound
 Reads raw data and outputs it as a wave file or image.
## How to build
```bash
cmake . -B build
cmake --build build --config Release
```
## How to use
In terminal give path to input folder and path to output folder: \
path/to/image/folder path/to/wave/folder
